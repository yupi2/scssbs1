SCSSBS - Simple Counter-Strike: Source Bunnyhop Script
======================================================

Install and follow the instructions for MSYS2.
https://msys2.github.io/

Install mingw32 and mingw64 with pacman in the MSYS2 shell.
	pacman -S mingw-w64-x86_64-clang mingw-w64-i686-clang mingw-w64-x86_64-tools mingw-w64-i686-tools

Use the command prompt and execute:
	build.bat
